# ⚡ Contact Form 邮件功能 - 快速开始

## 🎯 5分钟快速配置指南

### 1️⃣ 注册 Brevo 账号（免费）
👉 访问：https://www.brevo.com/
- 点击 "Sign up free"
- 填写邮箱和密码
- 验证邮箱

### 2️⃣ 获取 API Key
1. 登录后，点击右上角头像
2. 选择 **SMTP & API**
3. 点击 **API Keys** 标签
4. 点击 **Generate a new API key**
5. 命名为 `SaltMine`
6. 复制生成的 Key（格式：`xkeysib-xxxxxx...`）

### 3️⃣ 配置环境变量
在项目根目录找到 `.env.local` 文件，添加：

```bash
# 邮件服务配置
BREVO_API_KEY="xkeysib-你复制的API密钥"
BREVO_FROM_EMAIL="noreply@saltmine.com"
```

### 4️⃣ 重启开发服务器
```bash
# 停止当前服务器 (Ctrl + C)
# 重新启动
npm run dev
```

### 5️⃣ 测试功能
1. 访问：http://localhost:3000/contact
2. 填写表单并提交
3. 检查 `372509446@qq.com` 是否收到邮件
4. 查看终端日志确认发送状态

## 🚨 如果没收到邮件

### 检查清单：
1. [ ] API Key 是否正确复制（没有多余空格）
2. [ ] `.env.local` 文件是否在项目根目录
3. [ ] 是否重启了开发服务器
4. [ ] 查看 QQ 邮箱的垃圾邮件文件夹
5. [ ] 检查终端是否有错误提示

### 开发模式（无需配置）
如果暂时不配置邮件服务：
- 表单仍然可以提交成功
- 邮件内容会打印到终端
- 不影响用户体验

## 🔧 生产环境配置

部署到 Vercel 时：
1. 进入 Vercel Dashboard
2. 选择项目 → Settings → Environment Variables
3. 添加相同的环境变量：
   - `BREVO_API_KEY`
   - `BREVO_FROM_EMAIL`
4. 重新部署项目
